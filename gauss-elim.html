Programming Gaussian Elimination

Carlos M. Torres
(Launched on 6 April 2014)

Abstract

Gaussian Elimination is a general algorithm applied to matrices in order to reduce them to row echelon form. This method is particularly useful when we want to solve systems of linear equations that can be written on the form Ax = b. This particular paper will present this method when a applied to a system Ax = b, where A is a n x n matrix and b âˆˆ R n . The pseudo-code that will be presented will be applied to the augmented matrix of this system, which will have dimension [n, (n + 1)].
